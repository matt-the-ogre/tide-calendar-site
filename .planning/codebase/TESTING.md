## Page Object Model

**Base Class:** `tests/pages/BasePage.ts`
- Wraps Playwright `Page` instance
- Provides: `goto()`, `waitForPageLoad()`, `getTitle()`, `waitForElement()`, `elementExists()`, `clickElement()`, `fillInput()`, `selectOption()`, `screenshot()`
- Common locators via getters: `container`, `mainHeading`, `footer`, `socialIcons`
- Logging via custom `Logger` from `tests/utils/logger.ts`

**Page Object Pattern:**

```typescript
export class HomePage extends BasePage {
  constructor(page: Page) {
    super(page);
  }

  // Locators as getter properties (lazy, using centralized SELECTORS)
  get stationSearchInput(): Locator {
    return this.page.locator(SELECTORS.STATION_SEARCH);
  }

  // Actions as async methods
  async enterStationSearch(query: string): Promise<void> {
    this.logger.info(`Entering station search: ${query}`);
    await this.stationSearchInput.fill(query);
  }

  // Assertions as async methods prefixed with "assert"
  async assertAutocompleteVisible(): Promise<void> {
    await expect(this.autocompleteDropdown).toBeVisible();
  }

  // Composite actions
  async submitValidForm(options?: { stationId?: string; year?: string; month?: string }): Promise<void> {
    const { stationId = TEST_DATA.VALID_STATION_ID, year = TEST_DATA.CURRENT_YEAR.toString(), month = '01' } = options || {};
    await this.enterStationSearch(stationId);
    await this.selectYear(year);
    await this.selectMonth(month);
    await this.clickGenerateButton();
  }
}
```

## Test Structure Pattern

**Suite Organization:**

```typescript
import { test, expect } from '@playwright/test';
import { HomePage } from './pages/HomePage';
import { TEST_DATA } from './config/test-config';

test.describe('Feature Name', () => {
  let homePage: HomePage;

  test.beforeEach(async ({ page }) => {
    homePage = new HomePage(page);
    await homePage.navigateToHome();
    // Optional: wait for dynamic content
    await homePage.waitForYearDropdownPopulated();
  });

  test.describe('Sub-feature', () => {
    test('should do something specific', async ({ page }) => {
      // Arrange
      await homePage.enterStationSearch('Point');
      // Act
      await homePage.waitForAutocompleteDropdown();
      // Assert
      const itemsCount = await homePage.getAutocompleteItemsCount();
      expect(itemsCount).toBeGreaterThan(0);
    });
  });
});
```

test('should have working Generate PDF button', async ({ page }) => {
  const firstButton = page.locator('.quick-generate-btn').first();

## Adding New Tests

**New E2E test for a feature:**
1. Create `tests/<feature-name>.spec.ts`
2. Import page objects from `tests/pages/`
3. Add any new selectors to `SELECTORS` in `tests/config/test-config.ts`
4. Add any new test data to `TEST_DATA` in `tests/config/test-config.ts`
5. If new page interactions are needed, add methods to existing page objects or create a new one extending `BasePage`

**New page object:**
1. Create `tests/pages/<PageName>.ts` extending `BasePage`
2. Define locators as getter properties using `SELECTORS`
3. Define actions as async methods with logging
4. Define assertions as async methods prefixed with `assert`

**New Python unit test:**
1. Add test class to `app/test_tide_adapters.py` (or create new file)
2. Follow `unittest.TestCase` pattern with `setUp()` method
3. Add to `run_tests()` suite builder if in same file
