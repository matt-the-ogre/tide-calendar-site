- Files: `app/routes.py` (lines 182-194, 307-318), `app/get_tides.py`

- Severity: **Medium**
- Fix approach: Refactor `get_tides.py` to expose a `generate_pdf(station_id, year, month, location_name)` function that can be called directly from `routes.py` without subprocess invocation. This eliminates startup overhead, enables proper exception handling, and simplifies the code.

- Severity: **Medium**
- Fix approach: Use Python's `tempfile.NamedTemporaryFile` or `tempfile.mkdtemp()` for intermediate files. Replace `subprocess.run(["rm", ...])` with `os.remove()`. Add try/finally cleanup.
