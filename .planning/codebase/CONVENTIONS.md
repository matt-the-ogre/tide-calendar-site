## Error Handling

**Python Error Handling Patterns:**

Use specific exception types in `except` clauses. Never bare `except:`.

```python

    with sqlite3.connect(DB_PATH) as conn:
        # database operations
except sqlite3.Error as e:
    logging.error(f"Database error: {e}")
    return []  # Return empty list as fallback
```

```python

    log_station_lookup(station_id)

    logging.error(f"Database error logging station {station_id}: {e}")
    return None  # Don't re-raise -- logging is not critical
```

```python
# Pattern 3: Retry with exponential backoff for external APIs
for attempt in range(max_retries):
    try:
        if attempt > 0:
            time.sleep(retry_delay * attempt)  # Exponential backoff: 0, 2, 4 seconds
        response = requests.get(url, params=params, headers=headers, timeout=30)
        if response.status_code in [502, 503, 504]:
            continue  # Retry gateway errors
        elif response.status_code == 200:
            return self.parse_response(response.text)
        else:
            return None  # Non-gateway errors don't retry
    except requests.exceptions.Timeout:
        if attempt == max_retries - 1:
            return None
        continue
```
